# Mindrot App Testing

QA testing documentation for the Mindrot mobile application.

## Testing Scope

The following areas are being tested:

- Authentication
- Onboarding
- Core features
- UI behaviour
- Error handling

## Repository Structure

test-plan → overall testing strategy  
test-cases → manual test cases  
bug-reports → discovered defects  
exploratory-testing → session notes
